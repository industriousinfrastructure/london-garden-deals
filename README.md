# London house-hunt

Private research tool — live Rightmove garden 2–3 bed shortlist (deal-scored) + a gentrification-fit area map. Static, self-contained pages built from HM Land Registry, OpenStreetMap, TfL and Rightmove data.

- **index.html** — ranked deal shortlist (filter/sort, links to Rightmove)
- **map.html** — area heat map (momentum, amenity, transport, price-drop layers)

Rebuild: run the pipeline in the parent project, then `python deploy_git.py`.
